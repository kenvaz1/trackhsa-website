# 🚀 Launch Checklist

Use this checklist to ensure your marketing website is ready for launch!

## ✅ Pre-Launch (Required)

### Technical Setup
- [ ] Dependencies installed (`npm install`)
- [ ] Site runs locally (`npm run dev`)
- [ ] No console errors in browser
- [ ] All sections load correctly
- [ ] Navigation works (desktop & mobile)
- [ ] Mobile menu opens/closes properly
- [ ] FAQ accordion expands/collapses
- [ ] Smooth scrolling to sections works

### Content Updates
- [ ] App Store link updated (2 locations in `src/App.jsx`)
- [ ] App name correct throughout site
- [ ] Meta description updated in `index.html`
- [ ] Page title updated in `index.html`
- [ ] Copyright year correct in footer

### Testing
- [ ] Test on iPhone/iOS Safari
- [ ] Test on Android/Chrome Mobile
- [ ] Test on tablet (iPad)
- [ ] Test on desktop (Chrome, Safari, Firefox)
- [ ] All links work correctly
- [ ] Images load properly (if added)
- [ ] Text is readable on all screen sizes

### Build & Deploy
- [ ] Production build succeeds (`npm run build`)
- [ ] No build warnings or errors
- [ ] Deployed to hosting platform
- [ ] Live site loads correctly
- [ ] HTTPS enabled (automatic on Vercel/Netlify)

- [ ] Custom logo added (replace Receipt icon)

### Marketing

## 📊 Post-Launch (Recommended)

### Analytics & Tracking
- [ ] Google Analytics installed
- [ ] Google Search Console verified
- [ ] Conversion tracking set up
- [ ] App Store link clicks tracked

### SEO
- [ ] Sitemap.xml created and submitted
- [ ] Robots.txt added
- [ ] Open Graph tags added for social sharing
- [ ] Schema.org markup added
- [ ] Alt text added to all images

### Performance
- [ ] Lighthouse audit run (aim for 90+ scores)
- [ ] Page load speed tested
- [ ] Mobile performance verified
- [ ] Images optimized (if added)

### Marketing
- [ ] Website URL added to App Store listing
- [ ] Shared on social media (Twitter, LinkedIn, etc.)
- [ ] Posted in relevant communities (Reddit, Product Hunt)
- [ ] Email announcement sent (if you have a list)
- [ ] Press release distributed (optional)

## 🔍 Quality Assurance

### Visual Check
- [ ] No layout breaks on any screen size
- [ ] Colors look good and are consistent
- [ ] Typography is readable
- [ ] Spacing looks balanced
- [ ] Animations are smooth (not janky)
- [ ] Buttons have hover states

### Content Check
- [ ] No spelling errors
- [ ] No grammar mistakes
- [ ] All facts are accurate
- [ ] Pricing information correct
- [ ] Contact information correct (if added)

### Functionality Check
- [ ] All internal links work
- [ ] All external links work
- [ ] Links open in correct window (external = new tab)
- [ ] Forms submit correctly (if added)
- [ ] No broken images
- [ ] No 404 errors
