## 🎯 Quick Start (5 Minutes)

### Step 1: Install & Run

```bash
cd website
npm install
npm run dev
```

Open `http://localhost:5173` in your browser to see your site! 🎉

### Step 2: Update Your App Store Link

Open `src/App.jsx` and search for:
```
apps.apple.com/app/your-app-id
```

Replace with your actual App Store URL (appears in 2 places).

### Step 3: Deploy to Vercel (Free)

1. Push to GitHub:
   ```bash
   git add website/
   git commit -m "Add marketing website"
   git push
   ```

2. Go to [vercel.com](https://vercel.com)
3. Click "New Project" → Import your repo
4. Click "Deploy"
5. **Done!** Your site is live in 30 seconds 🚀

### Common Issues

**Site won't start?**


**Build fails?**

- Verify base URL in vite.config.js
- Clear browser cache
