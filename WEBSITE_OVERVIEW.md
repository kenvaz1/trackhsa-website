## Color Scheme

### Primary Colors
- **Brand Blue:** #0284c7 (buttons, accents)
- **Light Blue:** #f0f9ff (backgrounds)
- **Dark Blue:** #0c4a6e (text accents)

### Neutral Colors
- **White:** #ffffff (main background)
- **Gray-50:** #f9fafb (section backgrounds)
- **Gray-900:** #111827 (text)

### Gradients
- **Hero Text:** Blue gradient effect
- **Download Section:** Blue to cyan gradient background

## Typography

### Headings
- **H1 (Hero):** 4-7rem, bold, gradient text
- **H2 (Sections):** 2.5-3rem, bold
- **H3 (Features):** 1.25rem, semibold

### Body Text
- **Large:** 1.25rem (hero subtext)
- **Regular:** 1rem (descriptions)
- **Small:** 0.875rem (footer)

## Spacing & Layout

### Container
- **Max Width:** 1280px (7xl)
- **Padding:** 1-2rem responsive
- **Sections:** 5rem vertical padding

### Grid Layouts
- **Features:** 3 columns on desktop, 2 on tablet, 1 on mobile
- **Categories:** 6 columns on desktop, 3 on tablet, 2 on mobile
- **Testimonials:** 2 columns on desktop, 1 on mobile

## Interactive Elements

### Buttons
- **Primary:** Blue background, white text, rounded-2xl
- **Secondary:** Blue border, blue text, rounded-2xl
- **Hover:** Darker shade + smooth transition

### Cards
- **Border:** Light gray, rounded-2xl
- **Hover:** Blue border + shadow lift
- **Animation:** Slide up on scroll

### Navigation
- **Desktop:** Horizontal links
- **Mobile:** Hamburger menu
- **Scroll:** Smooth scrolling to sections

## Animations

### On Load
- **Hero:** Fade in
- **Features:** Slide up (staggered)
- **Categories:** Slide up (staggered)

### On Hover
- **Cards:** Border color change + shadow
- **Buttons:** Background color change
- **Links:** Color change

### On Interaction
- **FAQ:** Smooth expand/collapse
- **Mobile Menu:** Slide down
- **Scroll:** Smooth scrolling
